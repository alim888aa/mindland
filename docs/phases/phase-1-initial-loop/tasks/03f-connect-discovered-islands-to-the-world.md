# Connect discovered islands to the world

Status: in-progress

Owner: primary-agent

Depends on: Create initial islands

## Outcome

Replace the fixed four-island prototype map with the signed-in user's real islands discovered during onboarding, and make those same islands drive daily check-in choices and questionnaires.

## Requested direction

If onboarding discovers only Sleep, Study, and Fitness, the revealed world and daily check-in must contain those islands rather than the prototype Health, Relationships, Work, and Learning set. Persist ownership and purpose in Convex, render variable island counts and names, and thoroughly verify onboarding through later check-in.

V1 uses a procedural starter-island system backed by an approved visual catalogue. The AI selects the palette and small terrain or prop details that fit each island's purpose. Store those selections so the same island renders deterministically on every device and restart.

## Done when

A fresh real account can complete onboarding with a deliberately chosen set of topics, see exactly those persisted islands on its map, reopen the app without losing them, and use those same islands in both conversational and guided daily check-ins. A second account cannot see them.
