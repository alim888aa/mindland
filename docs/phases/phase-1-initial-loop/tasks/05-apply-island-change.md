# Apply growth, rocks, sinking, and resurfacing

Status: planned

Owner: unassigned

Depends on: Build daily check-ins

## Outcome

Translate recorded activity into visible positive nodes, expanding land, richer terrain features, accumulated negative points, rocks, gradual sinking, and possible resurfacing.

## Done when

Each island receives at most one summarized positive node, one positive point, and one negative point per day. Each positive node adds a small contextual detail from the approved catalogue. Every five positive points expands land or adds a larger contextual feature. Every five negative points creates a rock. A submerged island resurfaces when lifetime positive points exceed its rocks. These rules are implemented independently from rendering, tested against agreed examples, and reflected through island area, trees, hills, paths, buildings, rocks, and water level.
